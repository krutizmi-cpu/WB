# WB Unit Economics

Готовый Streamlit-проект для массового расчёта юнит-экономики по Wildberries.

## Что умеет
- загружает Excel со списком товаров;
- автоматически определяет категорию/предмет WB по названию товара;
- подтягивает комиссию по справочнику КВВ WB;
- считает СПП, рекламу, выкуп, брак, логистику, хранение, эквайринг и налог;
- выгружает горизонтальный Excel-отчёт.

## Что обязательно в шаблоне
Обязательные колонки:
- Артикул
- Наименование
- Длина, см
- Ширина, см
- Высота, см

Поддерживаются необязательные колонки:
- Вес, кг
- Себестоимость, ₽
- Цена продавца до СПП, ₽

Если необязательных колонок нет, приложение берёт значения по умолчанию из `config.json`.

## Как запустить
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Как настраивать экономику
Все бизнес-допущения вынесены в `config.json`:
- модель продаж (`FBW` / `FBS`);
- СПП;
- реклама;
- выкуп;
- брак и глубина потерь от брака;
- логистика и хранение;
- налоги;
- fallback-комиссия;
- коэффициенты склада.

## Важные допущения
1. Комиссия определяется по PDF-таблице КВВ WB через авто-сопоставление названия товара с предметом из справочника.
2. Если категория не определена уверенно, используется `fallback_commission_percent`.
3. СПП в модели считается как отдельный аналитический слой: показывается цена до и после СПП и сумма СПП.
4. Стоимость невыкупа считается как ожидаемая нагрузка на одну успешную продажу через показатель выкупа.
5. Стоимость брака считается как ожидаемый убыток = `себестоимость × % брака × % потери от брака`.
6. Формула ОСНО в этом проекте задана упрощённо через ставку из `config.json`, чтобы её можно было быстро адаптировать под ваш внутренний подход.

## Источники
- Раздел «Тарифы» на портале продавца WB: https://seller.wildberries.ru/instructions/ru/ru/material/fees-site-section
- PDF таблица КВВ для продавцов РФ, вступает в силу с 22 декабря 2025 года: https://static-basket-02.wbbasket.ru/vol20/portal/education/instruction/Koeffitsient_voznagrazhdeniia_Vaildberriz.pdf

## Состав проекта
- `app.py` — основное приложение
- `config.json` — скрытые бизнес-настройки
- `data/wb_kvv_lookup.csv` — подготовленный справочник КВВ WB
- `templates/wb_template.xlsx` — шаблон загрузки
- `requirements.txt` — зависимости
- `.streamlit/config.toml` — базовая настройка интерфейса

## Что можно улучшить дальше
- подключить живую выгрузку актуальных тарифов из кабинета продавца или внутреннего файла;
- обучить более точный классификатор категорий на вашем ассортименте;
- добавить рекомендованную цену под целевую маржу.
